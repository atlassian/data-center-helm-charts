# confluence

![Version: 0.14.0](https://img.shields.io/badge/Version-0.14.0-informational?style=flat-square) ![Type: application](https://img.shields.io/badge/Type-application-informational?style=flat-square) ![AppVersion: 7.12.3-jdk11](https://img.shields.io/badge/AppVersion-7.12.3--jdk11-informational?style=flat-square)

A chart for installing Confluence Data Center on Kubernetes

For installation please follow [the documentation in the repository](https://atlassian-labs.github.io/data-center-helm-charts/).

**Homepage:** <https://www.atlassian.com/software/confluence>

## Source Code

* <https://github.com/atlassian-labs/data-center-helm-charts>
* <https://bitbucket.org/atlassian-docker/docker-atlassian-confluence-server/>

## Requirements

Kubernetes: `>=1.19.x-0`
